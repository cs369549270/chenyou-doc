# 红包信息接口增加参数

接口地址

询问安卓开发获取

参数说明

参数 类型 说明
shanHuFlag int 是否还可以做珊瑚任务提额，根据taskType= 25 作为 key 查询 系统参数 growRedPackBase 中对应的 每天可完成总次数（totalNum）；
统计今天已做珊瑚任务提额的次数：统计 user_grow_pkg_line 表中 taskType= 25，日期是 今天 的 记录数；
若已完成次数 < totalNum ，则返回 shanHuFlag = 0 ,否则返回 1
taskListDone int 是否已完成每日任务，根据taskType= 27 作为 key 查询 系统参数 growRedPackBase 中对应的 param 中的 任务清单标识（code）；
根据code， uid 查询 任务清单表（activity_user_list）中 status= 0 ，valid_flag = 1 的记录，若记录存在，则返回 taskListDone = 0 ，否则返回 1
taskListCode string 每日任务的任务清单标识，根据taskType= 27 作为 key 查询 系统参数 growRedPackBase 中对应的 param 中的 任务清单标识（code）
addAmt string 补充任务增加的金额，上述【补充任务】给用户提交的金币数值 / 100
