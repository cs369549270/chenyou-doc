# 2.任务赚详情页，我的任务赚修改：

  • 添加返回参数“可再赚n元”：

计算方式：后台配置的邀请用户首单任务赚奖励 + 每单任务赚分成 * 当前任务的金额

如：后台配置的邀请用户首单任务赚奖励为 1.5 元，当前用户为普通用户，一级粉丝分成为 10%，任务金额为 5 元 ——> 1.5+10%*5=2元 （ps：团长，超级团长的分成比例各有不同）

  • 需要添加这个参数的接口：
  1. my/app-info/status?page=0&size=20&search=status:02,uid:999409
  2. my/app-info/dl/146
