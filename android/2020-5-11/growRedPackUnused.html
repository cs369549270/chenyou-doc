# 相关接口说明

# 1. 获取红包信息

请求参数： 登录状态

返回示例

{
   "base":"0.3",
   "videoNum":1,
   "unLockNum":3,
   "videoTaskNum":1
   "checkStatus":0,
   "gameStatus":0,
   "today":{
       "status":1,
       "amount":"0.3"
   },
   "tomorrow":{
       "status":0
       "amount":"0.7"
   }
}

返回参数说明

参数名 子参数名 类型 说明
base string 红包基数，用于【后天红包】的显示金额
videoNum int 领取【今天红包】已看视频次数
unLockNum int 领取【今天红包】需要看视频的次数
videoTaskNum int 今天看视频提额次数
checkStatus int 打卡任务状态
gameStatus int 玩小游戏任务状态
today json 【今天红包】信息
status string 今天红包状态：
0表示 可领取
1 表示 已领取
2 表示 未预约
amount string 今天红包的金额
tomorrow json 【明天红包】信息
status string 明天红包状态：
0表示 不可操作
1 表示 可提额
2 表示 未预约
amount string 明天红包的金额

# 2. 获取任务列表

请求参数：

参数名 类型 说明
uid int 用户标识
openCalendar int 是否已开启日历提醒
1 表示 已开启
0 表示未开启
openPush int 是否已开启推送
1 表示 已开启
0 表示未开启

返回示例

[{
   "icon":"http:i.shihuowang.com/ui/123.png",
   "title":"看视频打卡提额",
   "extraNumDesc":"今日还可打卡次数",
   "prizeDesc":"可提 0.02 ~ 0.3 元",
   "btDesc":"马上提额",
   "showNum":1,
   "totalNum":6,   
   "extraNum":1,
   "isCount":1,
   "count":123456,
   "status":"1",
   "taskType":"18",
   "emphasis":1,
   "param":{
       "id":"941212470"
   }
},{
   "icon":"http:i.shihuowang.com/ui/123.png",
   "title":"首页“点击赚钱”做任务",
   "extraNumDesc":"完成任务提额，次数不限",
   "prizeDesc":"可提 0.03~0.5 元",
   "btDesc":"马上提额",
   "showNum":0,
   "totalNum":6,
   "extraNum":1,
   "count":0,
   "status":1,
   "taskType":"19",
   "emphasis":1,
   "param":{}
}]

返回参数说明

参数名 子参数 类型 说明
icon string 图标
title string 粗体标题
extraNumDesc string 次数描述
prizeDesc string 奖励描述
btDesc string 任务状态为 0 时，按钮显示的文字
showNum int 是否显示“总共可领次数”和“剩余可领次数”
0 表示 不显示
1 表示 显示
totalNum int 总共可领次数
extraNum int 剩余可领次数
count int 倒计时剩余时间，单位：秒
status int 状态：
0 表示 待完成
1 表示 可领取，点击调提额接口，刷新页面
2 表示 倒计时中，按钮根据count显示倒计时
3 表示 明天再来
taskType string 任务类型,决定未完成状态下的点击效果，以及调【完成任务】接口时的传参
18 ：打开穿山甲激励视频 id：param.id，调【完成任务】接口
19 ：跳转首页
11 ：打开穿山甲激励视频 id：param.id，调【完成任务】接口
01 ：跳转【赚钱-小游戏】
20 ：webview形式打开推啊广告 id：param.id，调【完成任务】接口
21 ：跳转打开开启日历事项提醒权限页面，根据所传的事项内容和每天提醒时间，设置事项提醒
22 ：弹窗：关注公众号弹窗
23 ：跳转打开推送权限页面
emphasis int 是否显示底色
0 表示 不显示
1 表示 显示
param json 相关的参数
id string 激励视频的代码位id，或推啊广告的广告位id
event string 设置日历事项提醒用的事项文本
time string 设置日历事项提醒用的事项开始时间，每天的几点提醒用户

# 3. 完成任务

请求参数：

参数名 类型 说明
uid int 用户标识
taskType int 任务类型
isFinish int 是否直接提额，0 不直接提额； 1 直接提额

返回示例 当isFinish = 1 时，会返回以下数据

{
   "prizeAmount":"0.02",
   "nowAmount":"0.32"
}

返回参数说明

参数名 类型 说明
prizeAmount string 本次提额金额
nowAmount string 当前红包金额
