# 邀请规则第二套

用户达到要求则直接进入下一阶段，不同阶段的奖励不同时享受

WARNING

这套邀请规则，仅适用于channel_id = 565 的用户

邀请账户

用户需独立出一个邀请账户，单位：元，下文所有邀请奖励都是进入邀请账户

邀请阶段说明

WARNING

  1. 下述表格中的数值由系统参数配置
  2. 完成每日任务：activity_user_list 中 有 list_type = 00 ,status = 1 的记录
  3. 团队总邀请：包括一级，二级，三级，四级...无限级裂变总人数
阶段 升阶段要求 奖励方式 进入下一阶段的方式
第一阶段 被邀请人中有 3 位 完成过每日任务 每有一个徒弟完成每日任务奖励 3.5 元 徒弟完成每日任务时，完成人数达到 3 个
第二阶段 被邀请人中有 23 位 完成过每日任务 每有一个徒弟完成每日任务奖励 2 元 徒弟完成每日任务时，完成人数达到 23 个
第三阶段 徒弟中 有 100 人完成过每日任务，二级邀请达到 300 人 达到条件一次性奖励 200 元 被邀请者注册绑定邀请关系时，或提交邀请码时，达到人数要求；
徒弟完成每日任务时，完成人数达到 100 人
第四阶段（初级团长） 徒弟中有 1000 人完成过每日任务，团队总邀请达 4000 人 徒弟完成游戏赚/任务赚 奖励师傅任务金额的 8 %，二级邀请者完成游戏赚/任务赚 奖励师傅任务金额的 4 % 被邀请者注册绑定邀请关系时，或提交邀请码时，达到人数要求；
徒弟完成每日任务时，完成人数达到 1000 人
第五阶段（高级团长） 徒弟中有 5000 人完成过每日任务，团队总邀请达 20000 人 徒弟完成游戏赚/任务赚 奖励师傅任务金额的 10 %，二级邀请者完成游戏赚/任务赚 奖励师傅任务金额的 5 %，三级及以下（到第五级）邀请者完成游戏赚/任务赚 奖励师傅任务金额的 2 % 被邀请者注册绑定邀请关系时，或提交邀请码时，达到人数要求；
徒弟完成每日任务时，完成人数达到 5000 人
第六阶段（渠道合伙人） 徒弟完成游戏赚/任务赚 奖励师傅任务金额的 13 %，二级邀请者完成游戏赚/任务赚 奖励师傅任务金额的 7 %，三级及以下（到第五级）邀请者完成游戏赚/任务赚 奖励师傅任务金额的 3 %

相关系统参数

参数名：inviteRule2

说明：
参数 类型 说明
stage int 阶段
activeNum int 徒弟中完成每日任务的要求人数，仅适用于第一、二阶段
target float 阶段目标金额，返回给客户端显示
perPrize float 徒弟完成每日任务，奖励师傅，仅适用于第一、二阶段
firstInviteNum int 要求徒弟人数
secondInviteNum int 要求二级邀请人数
totalInviteNum int 要求团队总邀请人数，包括徒弟无限裂变人数，即包括一级，二级，三级，四级...二十级裂变人数
firstBonus int 徒弟完成游戏赚/任务赚，奖励师傅的任务金额分红比例，单位：%
secondBonus int 二级邀请用户完成游戏赚/任务赚，奖励师祖的任务金额分红比例，单位：%
moreBonus int 三级及以下邀请用户完成游戏赚/任务赚，奖励师祖的任务金额分红比例，单位：%
moreBonusLevel int 三级以下的分红，需要计算到第几级
