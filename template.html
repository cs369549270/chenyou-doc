# 领取任务

需求类型：

  • 修改

流程：

请求URL：

  • /m/confirm/download?userid=29474&appid=851&vcode=468D6B7090D89493F190AC8188DDAC4A

请求参数：

参数名 类型 说明
userid int 用户标识
appid int 任务标识
vcode string

流程说明：

流程 操作说明
任务状态 根据appid 查询 app_task_info 表中的 status
可领取数量 1. 根据appid 查询 app_task_plan 表中的 start_time <= now() 的 release_num 总和 → 已投放份数
2. 根据appid 查询 user_attend_task_line 表中的 status != '06' 的记录数 → 进行中份数
3. 已投放份数 - 进行中份数 → 可领取数量
新增参与记录 1. 在user_attend_app_line表中新增记录，status = '01'
2. 在user_attend_task_line 表中新增记录，status = '01'

返回示例

   {
       meta: {
           success: true, 
           message: "ok"
       }, 
       data: null, 
       extraData: null
   }

返回参数说明

参数名 类型 说明
success Boolean true 操作成功 ； false 操作失败
message string 服务端反馈

备注

&search=status:00 表示未参与 &search=status:01 表示进行中 &search=status:02 表示完成 &search=status:03 表示已参与
