# 管理后台审核任务上架

需求类型：

  • 优化

流程：

请求URL：

  • /app-task/audit-seller-task

请求参数：

参数名 类型 说明
id int 任务标识
status string 审核状态，通过传'1'

流程说明：

流程 操作说明
任务状态 根据appid 查询 app_task_info 表中的 status
409 任务状态非待审核，请刷新页面 状态码返回409，提示：“任务状态非待审核，请刷新页面”
修改任务状态为 1 根据appid 修改 app_task_info 表中的status 为 1
推送通知 通过个推推送新任务发布消息给用户
